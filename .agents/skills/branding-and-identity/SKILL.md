# UI/UX Master Class: Branding & Identity

Design is not just making things "look pretty". It is the strategic communication of a brand's core identity.

## ✅ ALWAYS

1.  **Define a Visual Language**:
    -   Is the brand playful? (Use rounded corners, vibrant colors, bouncy animations).
    -   Is it a corporate financial tool? (Use sharp corners, muted blues/grays, strict grid alignment, conservative animations).
    -   Every UI decision MUST trace back to the brand's core values.

2.  **Design System First**:
    -   Before building pages, establish a Design Token system: Primary Colors, Secondary Colors, Semantic Colors (Success, Error, Warning), Typography Scales, and Spacing Tokens.

3.  **Logo Placement & Breathing Room**:
    -   Always respect the logo's "clear space". Do not cram the logo next to a navigation link.

## ❌ NEVER

-   **Dilute the Brand Color**: Never use the primary brand color for everything. If the brand is Red, do not make error states, primary buttons, and active links the exact same Red. The user will lose context.
-   **Inconsistent Voice**: Never use playful copy (e.g., "Oopsie! Something broke 😅") in a UI that features a strict, corporate visual identity. Visuals and copy MUST match.
